## 2026-07-13

### Windows 安全自启动

- 新增无 `--accept-hooks` 的安全 Gateway 启动脚本，避免开机后自动批准未知 Shell Hook。
- 新增 Windows 登录自启动安装脚本，可创建快捷方式并选择立即启动 Gateway。
- 新增无密钥 `.env.example`，记录飞书、模型、数据目录和本地代理配置字段。
- 更新启动与故障排查文档，明确本机模式依赖开机、登录、联网和不休眠。

### 资讯个性化与反馈闭环

- 新增 SQLite 资讯历史，成功推送后记录条目，默认 14 天内不重复推荐。
- 为每条资讯生成稳定短 ID，支持有用、不相关、已知、稍后读四种显式反馈。
- 新增可编辑关注主题、历史窗口和单来源上限，排序综合热度、主题命中和来源反馈。
- 来源偏好设有上下限，避免反馈不断放大导致信息茧房。
- 新增 `profile` 查看画像与反馈统计，并补充 Hermes 指令路由说明。
- 新增白名单部署脚本，覆盖在线文件前自动备份，保留扩展版 `feishu_api.py`。
- 新增历史去重、反馈学习和兴趣排序测试。

### 飞书 CLI 可靠性与个人助手基础能力

- 补齐 `date_utils.py`、`requirements.txt` 和无敏感信息的 `config.example.yaml`，新增 `doctor` 配置检查。
- 飞书凭据改为请求发生时校验，使模块可以离线导入和测试；HTTP 错误不再静默伪装成正常响应。
- AI 资讯接入 HN、GitHub、ArXiv、Dev.to、Lobsters 五个真实来源，增加跨源去重和来源配额。
- 模型只返回候选 ID、译名和价值说明，链接、来源和热度由代码回填，避免生成错误链接。
- 外部资讯明确按不可信数据处理，代理地址改为可选环境变量。
- 技能训练会话改为按用户持久化，修复任务拆解未保存四轮回答的问题。
- 新增日期解析、资讯链接保真/去重/多样性和训练会话测试。
- 文档明确当前由 Hermes + DeepSeek 驱动，Codex 不在在线消息运行链路中。

## 2026-07-08

### 航海手册 MCP 工具链适配

- 查询结果新增“可检索资料”过滤：待审核、低价值、关键壳子缺失记录不会进入搜索结果。
- 无结果搜索词会写入 `data/no-result-searches.jsonl`，用于运营后续补资料。
- 新增 `npm run doctor`，一键检查环境变量、核心语法、资料库体检、MCP 航海手册 dryRun 和知识包导出。
- 新增 `npm run export:knowledge`，可将多维表格资料导出为 Markdown 和 JSONL 知识包，便于导入 MaxKB/Dify/RAG 系统。
- 新增 `npm run audit:library` 和 `audit_library` 工具，支持只读资料库体检。
- 资料库体检会统计总记录、可检索完整度、MCP 占比、待审核、重复指纹、壳子缺失和低价值候选。
- 机器人新增“资料库体检/检查资料库/看看资料库健康度”意图，可返回健康报告。
- 新增 `npm run verify:mcp-manual`，用于一键验证生财 MCP 航海手册三件套和 dryRun 读取链路。
- 更新航海小抓 README，补充运行、验证、私聊指令和 MCP 手册同步配置。
- 为航海小抓新增生财 MCP 目录型工具链同步能力。
- 支持按“航海列表 → 航海手册章节列表 → 航海手册章节详情”读取章节内容，并写入现有多维表格资料库。
- 保留原有关键词搜索型 MCP 同步能力，不需要人工上传表格。
- `.env.example` 新增航海手册同步开关和数量限制配置。
- 新增私聊确定性指令：“同步航海手册”“拉取航海手册”“更新航海手册”，专门触发航海手册 MCP 工具链。
