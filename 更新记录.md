# 更新记录

## 2026-07-08

### 航海手册 MCP 工具链适配

- 新增 `npm run export:knowledge`，可将多维表格资料导出为 Markdown 和 JSONL 知识包，便于导入 MaxKB/Dify/RAG 系统。
- 新增 `npm run audit:library` 和 `audit_library` 工具，支持只读资料库体检。
- 资料库体检会统计总记录、可检索完整度、MCP 占比、待审核、重复指纹、壳子缺失和低价值候选。
- 机器人新增“资料库体检/检查资料库/看看资料库健康度”意图，可返回健康报告。
- 新增 `npm run verify:mcp-manual`，用于一键验证生财 MCP 航海手册三件套和 dryRun 读取链路。
- 更新航海小抓 README，补充运行、验证、私聊指令和 MCP 手册同步配置。
- 为航海小抓新增生财 MCP 目录型工具链同步能力。
- 支持按“航海列表 → 航海手册章节列表 → 航海手册章节详情”读取章节内容，并写入现有多维表格资料库。
- 保留原有关键词搜索型 MCP 同步能力，不需要人工上传表格。
- `.env.example` 新增航海手册同步开关和数量限制配置。
- 新增私聊确定性指令：“同步航海手册”“拉取航海手册”“更新航海手册”，专门触发航海手册 MCP 工具链。
