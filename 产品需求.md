# 产品需求

## 背景

本项目用于建设公司内部资料抓取与 AI 整理能力。

## 目标

- 自动抓取指定资料来源。
- 对资料进行清洗、归类、摘要与结构化输出。
- 支持通过 Agent 或 MCP 工具链复用能力。
- 保留可追溯的执行日志与测试记录。
- 接入飞书 CLI 能力，用于后续对接飞书相关工具链。
- 接入航海小抓能力，用于资料抓取场景的最小可运行原型。

## 待明确问题

1. 资料来源有哪些，是否需要登录或授权？
2. 抓取结果需要输出到哪里？
3. 是否需要人工审核流程？
4. 是否有合规、隐私或访问频率限制？
5. 飞书 CLI 与航海小抓源码当前存放在哪里？
